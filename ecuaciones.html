📐 ¿Qué es una ecuación y cómo se clasifica?

Una ecuación matemática es una igualdad entre dos expresiones que contiene una o más incógnitas. Resolver una ecuación significa encontrar el valor (o valores) de esas incógnitas que hacen verdadera la igualdad. En matemáticas, física e ingeniería encontrarás distintos tipos, y elegir el método correcto marca la diferencia entre una solución exacta en segundos y un cálculo que no converge:

LINEALES     → ax + b = 0         → siempre 1 solución real: x = −b/a
CUADRÁTICAS → ax² + bx + c = 0    → hasta 2 soluciones (reales o complejas)
CÚBICAS/CUÁRTICAS → grado 3 ó 4     → resolución numérica o analítica
TRASCENDENTES → sin, cos, ln, exp → solo resolución numérica
SISTEMAS LINEALES → varias ecuaciones, varias incógnitas

La herramienta de arriba cubre todos estos casos: usa la pestaña Sistemas Lineales para sistemas 2×2 y 3×3, Polinómica para ecuaciones de grado 2 a 4, y Ecuación Libre para cualquier expresión con sin, cos, ln, exp y más.

🔢 La Regla de Cramer: cómo resolver sistemas de ecuaciones

La Regla de Cramer es el método algebraico estándar para resolver sistemas de ecuaciones lineales cuando el número de ecuaciones iguala al número de incógnitas. El procedimiento es el siguiente:

1. Construye la matriz de coeficientes [A] y el vector de términos independientes [b].
2. Calcula el determinante principal D = det(A).
3. Si D = 0, el sistema es incompatible (sin solución) o indeterminado (infinitas soluciones).
4. Para cada variable xᵢ, reemplaza la columna i de [A] con el vector [b] para obtener la matriz Dᵢ.
5. La solución es: xᵢ = Dᵢ / D

Ejemplo 2×2:   2x + 3y = 8    /    5x + y = 7
D = 2×1 − 3×5 = −13
Dx = 8×1 − 3×7 = −13   → x = Dx/D = 1
Dy = 2×7 − 8×5 = −26   → y = Dy/D = 2

La aplicación calcula y muestra cada determinante intermedio (D, Dx, Dy, Dz) para que puedas seguir el proceso paso a paso, ideal para verificar tus cálculos manuales o entender el método en álgebra lineal.

📈 La fórmula cuadrática y el discriminante

Para ecuaciones de grado 2 de la forma ax² + bx + c = 0, la herramienta aplica la fórmula cuadrática general, también llamada fórmula de Bhaskara:

x = ( −b ± √(b² − 4ac) ) / 2a

Discriminante D = b² − 4ac:
  D > 0 → dos raíces reales distintas
  D = 0 → una raíz real doble: x = −b / 2a
  D < 0 → dos raíces complejas conjugadas: a+bi y a−bi

Ejemplo: x² − 5x + 6 = 0 → a=1, b=−5, c=6. Discriminante: D = 25 − 24 = 1. Soluciones: x = (5 ± 1) / 2, por tanto x₁ = 3 y x₂ = 2. Como D > 0, ambas raíces son reales y distintas.

⚡ El método Newton-Raphson: cómo funciona

El método de Newton-Raphson es el algoritmo iterativo de búsqueda de raíces más rápido para funciones diferenciables. Su intuición geométrica es elegante: partiendo de un punto inicial x₀, traza la recta tangente a la curva f(x) y encuentra dónde corta el eje x (ese punto es x₁). Repite el proceso desde x₁ y así sucesivamente hasta converger:

x_(n+1) = x_n − f(x_n) / f'(x_n)

Esta app usa derivada numérica:
f'(x) ≈ [ f(x+h) − f(x−h) ] / (2h)    con h = 10⁻⁷

Ventajas: convergencia cuadrática (el número de decimales correctos se duplica en cada iteración). Desventajas: puede no converger si el punto inicial está lejos de la raíz o si la derivada se anula cerca de la solución. Por eso, esta herramienta combina Newton-Raphson con Bisección como respaldo: si Newton falla, Bisección toma el relevo garantizando la convergencia.

🎯 El método de bisección: garantizando convergencia

El Teorema de Bolzano establece que si f(a) y f(b) tienen signos opuestos en una función continua, entonces existe al menos una raíz en el intervalo [a, b]. El método de bisección explota este hecho de manera sistemática:

1. Toma el punto medio m = (a + b) / 2.
2. Evalúa f(m): si |f(m)| < tolerancia, m es la raíz.
3. Comprueba en qué mitad se produce el cambio de signo y repite.
4. El intervalo se reduce a la mitad en cada iteración → convergencia garantizada.

Es más lento que Newton-Raphson (convergencia lineal vs. cuadrática), pero siempre converge. La aplicación escanea automáticamente 500 subintervalos dentro del rango [a, b] que introduzcas, buscando cambios de signo para detectar todas las raíces reales posibles, no solo una.

🗺️ Cuándo usar cada método: guía práctica

¿No sabes qué pestaña elegir? Aquí tienes la guía rápida:

Sistema 2 ó 3 incógnitas con coeficientes → Regla de Cramer
Ecuación de grado 2 (ax²+bx+c=0)     → Cuadrática (exacto)
Polinomio de grado 3 ó 4             → Durand-Kerner numérico
Ecuación con sin/cos/ln/exp          → Ecuación Libre
No sabes el tipo                     → Ecuación Libre, siempre funciona

Para el método libre, elige una semilla (x₀) cercana a la raíz que sospeches y un rango de búsqueda [a, b] razonable. Si la función tiene varias raíces en ese intervalo, la aplicación las encontrará todas automáticamente.

❓ Preguntas frecuentes

¿Qué significa que un sistema sea "incompatible"?
Significa que las ecuaciones son contradictorias: no existe ningún valor de las incógnitas que satisfaga todas simultáneamente. Matemáticamente, el determinante D = 0. Si además los determinantes Dᵢ también son cero, el sistema es indeterminado (infinitas soluciones, ecuaciones redundantes). La herramienta detecta ambos casos automáticamente.

¿Por qué a veces Newton-Raphson no converge?
El método necesita que el punto inicial x₀ esté suficientemente cerca de la raíz real. Si la función tiene pendiente muy pequeña (derivada ≈ 0) en ese punto, o si x₀ cae en una zona donde la función oscila, el método puede divergir o ciclar. Por eso la aplicación siempre complementa Newton-Raphson con el escaneo por bisección en el rango que especificas, asegurando que se encuentren raíces aunque Newton falle.

¿Puedo resolver sistemas con más de 3 incógnitas?
Esta herramienta está optimizada para sistemas 2×2 y 3×3. Para sistemas más grandes (4×4 o superior) se recomienda usar la eliminación gaussiana (reducción por filas) o factorización LU, técnicas propias del álgebra lineal numérica que exceden el alcance de esta calculadora de bolsillo.

¿Qué son las raíces complejas y para qué sirven?
Son números de la forma a + bi, donde a es la parte real, b la parte imaginaria e i = √(−1). Aparecen cuando el discriminante D < 0 en una ecuación cuadrática, o en polinomios de grado superior sin todas las raíces reales. En ingeniería eléctrica y análisis de señales, las raíces complejas describen oscilaciones y respuestas de sistemas dinámicos. La herramienta las muestra en formato a ± bi.
